# CloudFin ISMS / Assurance Portfolio (ISO/IEC 27001)

This repository contains a structured, evidence-based internal ISMS / assurance portfolio for a B2B SaaS company scenario (CloudFin Analytics).

## Structure
- 01_ISMS_Core: ISMS scope, policy, risk methodology
- 02_Registers: Risk Register, Statement of Applicability (SoA)
- 03_Audit_Pack: Audit plan, checklist, evidence log, findings, CAPA tracker
- 04_Evidence: Evidence artifacts and Evidence Index (EV-###)
- 05_Exports_Final_PDF: Final exported portfolio PDFs

## Evidence Rule
Every SoA control and audit finding must reference an Evidence ID (EV-###) listed in the Evidence Index.
- Documents follow ID/version/date naming.
